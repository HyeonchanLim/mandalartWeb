## 프로젝트 목적
오타니 쇼헤이의 성공비결로 언급되면서 화제가 되었고, 이 계획표를 통해 개발 계획 , 성장 계획 등 여러 분야에서 사용이 가능하다 판단되었습니다.
그리고 기존의 단순한 Todo List 보다 좀 더 시각적인 목표 설정 도구를 제시하는 프로그램이라 생각하여 선정하였습니다.
